# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it **privately** so it can be fixed safely before public disclosure.

* **Email:** [mail@kf637.com](mailto:mail@kf637.com?subject=Discord%20Support%20Ticket%20Bot%20Vulnerability%20Report)  
* **PGP Key (optional):** [PGP key link or fingerprint]

Please include the following details in your report:

1. Description of the vulnerability  
2. Steps to reproduce  
3. Potential impact  
4. Environment (OS, Node.js version, Discord.js version, etc.)

I will respond within **7 days** and aim to resolve security issues as quickly as possible.

## Security Updates

* Updates will be applied to the `main` branch  
* Users should pull the latest changes regularly to ensure they have the most secure version  
* Critical issues may be backported manually if needed  

## Reporting Guidelines

Do **not** open a public issue for security concerns, as this may expose users to risk. Always contact me directly via the private channel above.
